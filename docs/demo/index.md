# Knowledge Base

Welcome to your shared context space. This is where you and your AI agents collaborate.

## Structure

| Folder | Purpose |
|--------|---------|
| **briefings/** | Daily briefings written by your agents |
| **lists/now/** | Weekly task lists and notes |
| **projects/** | Active and completed project folders |
| **thoughts/** | Ideas and backlog items to explore |
| **libraries/** | Shared external libraries (each its own repo) |
| **meta/** | Templates and operating protocols |

## Getting Started

Your agents write to this knowledge base — briefings, task updates, project status. You review, adjust, and steer through Hilt's views or by editing these files directly.

The **Briefing** tab shows today's briefing. The **Bridge** tab shows your weekly tasks and project board. This **Docs** tab lets you browse and edit everything.
