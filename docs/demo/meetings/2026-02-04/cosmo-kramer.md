# Coffee Table Book Website Kickoff

Kramer wants a website for his coffee table book about coffee tables. The twist: the website itself should look like a coffee table. He brought sketches on napkins. Some of them were actually good.

Key features discussed:
- 3D book viewer with page-flip animation
- "Place on your table" AR feature (ambitious)
- Celebrity endorsement carousel (he claims he has commitments)

We agreed to start with a simple landing page and iterate from there.
