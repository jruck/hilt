# Neighbor Analysis Task

## Objective

Research and analyze 50 neighbors in the AI coding agent session/project management space to:

1. Map Hilt's market position and unique value proposition
2. Identify 100 feature improvement opportunities
3. Understand where the market is heading and where Hilt should focus

Build on existing research in `docs/neighbors/` which has 6 tools already analyzed.

## Success Criteria

- [ ] `docs/neighbor-analysis/README.md` exists with full report
- [ ] `docs/neighbor-analysis/matrix.md` contains exactly 50 competitors
- [ ] `docs/neighbor-analysis/features.md` contains exactly 100 feature ideas
- [ ] All sections present: Executive Summary, Methodology, Comparison Matrix, Category Breakdown, Feature Gap Analysis, Positioning Recommendations
- [ ] Each competitor entry includes: name, URL, category, pricing, key differentiator
- [ ] Features are prioritized as must-have, should-have, or nice-to-have

## Scope

### In Scope

- AI coding assistants with session/project management
- Agent orchestration and multi-agent tools
- AI task management tools
- Developer productivity tools with AI conversation management
- Session managers for any AI coding tool
- Open source and commercial products

### Out of Scope

- Pure chat interfaces without session concepts
- AI coding assistants themselves - we complement them not compete
- Non-developer AI tools

## Research Strategy

1. Start with existing research - Import from docs/neighbors/ which has 6 tools
2. Expand from known competitors - Weft, Conductor, Nimbalyst, Vibe Kanban, Claude Code UI, Lightsprint
3. Search GitHub for: claude code, ai session manager, coding agent, agentic workflow, ai kanban
4. Search Product Hunt for AI developer tools
5. Search HN and Reddit for AI coding workflow tools
6. Check YC directory for AI and developer tools
7. Check VS Code and JetBrains marketplace for AI extensions
8. Find Awesome lists for AI coding tools

## Output Files

### README.md - Full Report

1. Executive Summary - market landscape and Hilt position
2. Methodology - how research was conducted
3. Competitor Matrix - 50 tools with key attributes
4. Category Breakdown
   - Session Managers - closest to Hilt
   - Agent Orchestrators
   - Task-Centric Tools
   - IDE Extensions
   - Cloud Platforms
5. Feature Gap Analysis
   - Features market has that Hilt lacks
   - Features Hilt has that market lacks
   - Emerging trends
6. Positioning Recommendations

### matrix.md - Comparison Matrix

Table with 50 rows, columns for:
- Name
- URL
- GitHub if available
- Category
- Pricing
- Key Differentiator
- Relevance to Hilt

### features.md - 100 Feature Ideas

Organized by category:
- Session management and organization
- AI and ML enhancements like summaries and search
- Terminal and execution
- Collaboration and sharing
- Integrations with MCP APIs and IDEs
- UX and UI improvements
- Data and analytics
- Unique and innovative ideas

Each feature marked as must-have, should-have, or nice-to-have.
