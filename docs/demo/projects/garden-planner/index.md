# Garden Planner

A tool for planning seasonal gardens with frost date calculations, companion planting recommendations, and a visual timeline.

## What Shipped

- **Frost date calculator** — Looks up USDA hardiness zone by zip code, calculates last/first frost dates, and suggests planting windows for common vegetables.
- **Companion planting matrix** — Color-coded grid showing which plants grow well together and which to keep apart. Data sourced from university extension services.
- **Seasonal timeline** — SVG-based view showing planting, growing, and harvest windows across the year.

## Tech

Simple static site. Vanilla JS + SVG for the timeline. No framework needed — the data is small and the interactions are straightforward.
