# Semantic layer v0.1 — initial build

The first end-to-end semantic knowledge layer, shipped flag-gated (`HILT_SEMANTIC_ENABLED`).

**What this generation is.** gemini-embedding-001@1536 embeddings over the main vault + saved
Library references (text-only, `libraries/` and `references/.cache` candidates excluded), four-bucket
Flash entity extraction (person/project/idea/source) with two-stage dedupe, and a UMAP/HDBSCAN
emergent hierarchical topic taxonomy labeled by the stronger Gemini Pro / Claude pass with split/
merge/birth/death lineage.

**What to review when a sample lands.** Spot-check the topic labels for practitioner voice and
specificity (not "AI" but "Autonomous agent architecture"); confirm entity resolution merged true
referential duplicates and abstained on merely-related pairs; confirm `related <path>` surfaces
serendipitous through-lines the explicit-link graph couldn't.

**Still open.** No integer baseline is published yet — this is the iteration lane. The re-fit cadence
(weekly) is env-tunable and may want tuning once real drift is observed; `sqlite-vec` ships OFF
(BLOB-cosine fallback) until the packaged-Electron dlopen path is verified.
