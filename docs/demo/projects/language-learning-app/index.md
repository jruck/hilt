# Language Learning App

A spaced repetition app that uses AI to generate contextual example sentences instead of isolated flashcards. The idea is that vocabulary sticks better when you encounter it in realistic situations.

## Open Questions

- What language(s) to start with?
- Mobile-first or web-first?
- How to handle pronunciation — TTS good enough, or need recorded audio?
- Is this different enough from Anki + ChatGPT to be worth building?

## Prior Art

- Anki — gold standard for SRS, but cards feel sterile
- Duolingo — gamified but shallow
- Clozemaster — closest to this idea, but no AI generation
