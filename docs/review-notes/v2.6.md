# Source description, not recommendation copy

This pass separates two jobs that had begun to blur together. The Library note's `description` now
explains only what the source contains; timing, active-work references, personal framing, and reasons
to pay attention belong to the recommendation episode instead.

The failure to check for is duplicated editorial copy: a source description that sounds like "read
this now," or a recommendation pitch that simply restates the description. This iteration tightens
newly processed descriptions and rejects near-duplicate pitches, but it does not rewrite otherwise
healthy older notes until they are naturally reprocessed.
