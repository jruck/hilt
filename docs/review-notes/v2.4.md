# Structured Reweave Output

This iteration changes the reweave transport contract, not the digest voice. Claude now returns the
digest, connections, and attention judgment through native JSON Schema output.

The motivating failure was **Notion Ship OS**: the agent completed its vault exploration and wrote a
useful result, but the phrase `"replace Notion with Loft"` appeared unescaped inside prompt-only JSON.
Hilt discarded the whole result and left the item at `Ready · Connections pending` while newer items
completed.

Review that quoted prose survives, connections and judgment persist, the pending marker clears, and
the worth score appears. Thin source capture remains a separate quality constraint; schema reliability
does not invent missing source detail.
