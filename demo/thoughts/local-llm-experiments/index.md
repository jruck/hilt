# Local LLM Experiments

Run smaller language models locally for tasks where latency matters more than capability. Explore what's possible with 7B–13B parameter models on Apple Silicon.

Potential uses:
- Real-time code completion (faster than API roundtrip)
- Local document summarization (no data leaves the machine)
- Quick classification tasks for the home automation rules engine

Start with Ollama and see what models fit in 16GB RAM.
